# Slash-Mark-APSCHE-Intern-Task-3
## Task 3 Completed – Password Generator

I have successfully completed **Task 3** of my APSCHE internship by developing a **Password Generator**.

### Project Overview

This project generates secure passwords based on user-selected length. It creates random combinations of letters, numbers, and special characters to help users generate strong passwords.

### Features Implemented

* User-defined password length
* Random password generation
* Combination of letters, numbers, and symbols
* Password strength support
* Option to regenerate password
* Exit option
* Simple interactive program

### Technologies Used

* Python
* Random module
* String module

### Outcome

This task improved my understanding of Python modules, string manipulation, randomization, and creating useful utility programs.
